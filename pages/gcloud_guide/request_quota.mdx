# Google Cloud Quotas

To deploy LLMs on Google Cloud, you need to be able to acess GPU resources from Google Cloud. Google Cloud Quotas determine if and how many (GPU) resources you can access. By default, your quotas will be very restrictive, so you will most likely have to request limit increases to deploy models.

Here are some helpful resources from Google Cloud:

- [About Quotas](https://cloud.google.com/docs/quota?hl=en)
- [Request a Limit Increase](https://cloud.google.com/docs/quota_detail/view_manage?hl=en#requesting_higher_quota)
